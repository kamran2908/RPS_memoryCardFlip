A detailed explanation about what this repository is about <br>

This is basically a game which has two round where a player who wins both the round gets a point<br>
round 1: <br>
This round is memory matrix wherein there is a 5x5 grid . In that certain boxed will highlighted and that is a pattern, this pattern will be displayed to player for few seconds so he can have a look at which boxes are highlighted . After that player has to select the highlighted boxes displayed previously. If player succeeds he is eligible to play the second round of the game
<br>
round 2:<br>
this round is Memory Card Flip where in you have a series of cards each card has some symbol (like emojis , fruits , animals ) pair of cards have same symbol . 
You have to pick only two cards if these cards have same symbol then u score a point else no
after completing this round you will be redirected to round 1 and this will be continued <br>
Player who scores 5 points wins the game.
